==================
Django Profile GAE
==================

This is a user private zone/profile management application for Google AppEngine running Django
, allowing the users of your application to insert information about him in his profile.

Inside this package you will find a demo application which will show
you what can be accomplished. You view an online version of this demo here:

http://django-profile.appspot.com

The application use Django and the Google AppEngine libraries on the backend, and Jquery
and a lot of plugins for the user frontend. These are the main plugins used:

* http://marcgrabanski.com/code/ui-datepicker/
* http://brandonaaron.net/docs/dimensions/
* http://code.google.com/p/jquery-checkbox/
* http://odyniec.net/projects/imgareaselect/
